# Apple Core 1.0

Apple Core is a native macOS menu bar app that lets MCP clients work with Apple services on your Mac. The standard build has 77 tools for Calendar, Reminders, Contacts, Location, Maps, Messages, screen and camera capture, Shortcuts, Notes, Mail, and utility tasks.

Connections use a bearer token that the app generates locally. New installs listen only on localhost. Remote access is optional, and each service stays local until you turn on its Remote toggle. If you choose to use a Cloudflare Tunnel, Apple Core creates configuration for your own Cloudflare account.

The download does not contain the developer's bearer token, Cloudflare credentials, OAuth clients, Apple account data, or macOS privacy permissions. Every Mac starts with its own configuration under `~/.config/apple-core/` and requests access to Apple services through the normal macOS permission controls.

Version 1.0 is signed with a Developer ID certificate, notarized by Apple, and distributed through a signed Sparkle update feed.
